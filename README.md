# talon-gis-arcgispro
Talon GIS: ArcGIS Pro voice commands

## Setup
1. Install Talon & Download Talon Community Repository
    https://talon.wiki/Resource%20Hub/Talon%20Installation/installation_guide

2. Install git:
    https://git-scm.com/install/

3. Download MouseFree repository to your talon user folder
    - Launch git CMD
    - At the prompt, enter: 	
	    `cd %appdata%\talon\user`
    or say:				
	    `'go talon user'`
    - At the prompt, enter:		
        `git clone https:/github.com/geobarry/mousefree.git`

4. Download talon-gis-arcgispro repository
    - Same steps as above but at the prompt enter:
        `git clone https://github.com/geobarry/talon-gis-arcgispro.git`
